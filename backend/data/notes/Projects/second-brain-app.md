# Project: Second Brain App

Building an #Obsidian-like note-taking application for personal knowledge management.

## Features

- Three-pane layout (sidebar, note list, editor)
- Graph view with force-directed layout
- Wiki-links [[Note Name]] support
- Auto-tagging with #tags

## Tech Stack

- React + TypeScript + Vite
- Express backend
- D3.js for graph visualization
- Tailwind CSS for styling

2. Add search functionality
